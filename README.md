Docsify turns one or more Markdown files into a Website, with no build process required.

## Process

- 일반 처방전 및 약봉투 사진을 입력한다.
- Naver General OCR을 돌려 얻은 raw text를 전처리하여 약품명이 포함된 것으로 추측되는 lines만 남긴다.
- Smart full-text search plugin
- Multiple themes
- Useful plugin API
- Emoji support
